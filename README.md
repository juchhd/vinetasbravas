# Viñeta Brava - Revista de Cómic Latino

Revista digital interactiva de cómic manga latino.

## Despliegue en GitHub Pages

1. Sube todos los archivos al repositorio
2. Ve a Settings > Pages
3. Selecciona la rama `main` o `master`
4. Guarda y espera unos minutos
5. Tu sitio estará disponible en: `https://tuusuario.github.io/nombre-repositorio/`

## Estructura de archivos

```
/
├── index.html
├── styles/
│   └── main.css
├── js/
│   └── script.js
├── data/
│   ├── catalog.json
│   └── pages.json
├── recursos/
│   ├── logo.png
│   ├── fondo.jpg
│   ├── catalogo.png
│   └── mascota.png
└── paginas/
    └── [archivos de páginas]
```
